The LiveCFD Project

Welcome to the LiveCFD project! This is a set of modules for Slackware Live Edition 15.0, that makes any x86_64 PC a KISS appliance for CFD in seconds.

How to install:

1. Create a persistent Slackware Live Edition 15.0 x86_64 on a bootable disk.

2. Download the modules you need:

LiveCFD Blender 2.93 - 389MB (md5 c6199d574ad9f75099298ce3ba1dd9b4): Default 'live' user bundled Blender version 2.93 with SnappyHexMesh GUI Add-on for preprocessing.

LiveCFD ParaView 5.10 - 486MB (md5 526b81f819304751acdae5eadd118c66): Default 'live' user bundled ParaView version 5.10 for postprocessing.

LiveCFD OpenFOAM Build Dependencies - 16MB (md5 0fcd37360c87259c60c8379dcf760148): System packages required for all OpenFOAM versions to compile.

LiveCFD OpenFOAM v8 - 217MB (md5 7f9fb6f8131265ae43a3ad671d9c45b6): Default 'live' user OpenFOAM version 8 installation.

LiveCFD OpenFOAM v8 C++ Source Code Guide - 485MB (md5 95b9fc7329206444f365672d50064963): OpenFOAM Version 8 documentation.

LiveCFD OpenFOAM v9 - 209MB (md5 c1951bc6c172b767f4a2bffb530ac552): Default 'live' user OpenFOAM version 9 installation.

LiveCFD OpenFOAM v9 C++ Source Code Guide - 490MB (md5 7b02419e15a054d80235d9faa21057b1): OpenFOAM Version 9 documentation.

LiveCFD OpenFOAM v10 - 251MB (md5 5d31a4708645bbf6540736087a0ee9ba): Default 'live' user OpenFOAM version 10 installation.

LiveCFD OpenFOAM v10 C++ Source Code Guide - 486MB (md5 3e85bbce3040fe2dfc45fecbcd5cae35): OpenFOAM Version 10 documentation.

LiveCFD OpenFOAM v11 - 230MB (md5 6166558ad1052d55a578e8a4b5d97376): Default 'live' user OpenFOAM version 11 installation.

LiveCFD OpenFOAM v11 C++ Source Code Guide - 522MB (md5 59cb21d727c913e47bc5d52738208715): OpenFOAM Version 11 documentation.

3. Copy the modules into the "liveslak/addons" subdirectory, and reboot, e.g.

live@darkstar:~$ sudo cp /home/live/Downloads/*.sxz /mnt/livemedia/liveslak/addons

Enjoy!
